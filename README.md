# Goals
Build a Laravel and Vue app using TDD.

The app will consist of a curated list of videos 
from past Laracons.

# Project Components
* Backend for CRUD
* Web scrapper to get videos
* API
* Front End

# User Stories
As a user, I want the ability to click on a speakers name
and see all of his or her talks on one page.

As a user, I want the ability to search all videos so I 
don't need to manually find what I'm looking for.

Mini Player

Admin CRUD
